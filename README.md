# pommes-h2-network

## Industry – Raw Data

Ce dossier contient les **données brutes (raw data)** utilisées pour la construction des jeux de données relatifs à la **demande industrielle en hydrogène** dans le cadre du projet **Pommes H₂ Network**.

Les données présentes ici servent de **base d’entrée** aux traitements, agrégations et scénarios développés dans les notebooks d’analyse, et ne correspondent pas nécessairement à des résultats finaux ou harmonisés.

### Contenu
- Données sectorielles industrielles (acier, aluminium, chimie, raffinage, etc.)
- Données issues de sources institutionnelles européennes (ex. Eurostat, TYNDP, statistiques énergie)
- Fichiers intermédiaires nécessaires à la reproductibilité des calculs

### Usage
Ces données sont destinées à être :
- utilisées pour des analyses de scénarios, modélisation énergétique ou études prospectives H₂

⚠️ **Attention** :  
Les données de ce dossier sont considérées comme *raw / non transformées*.  
Elles peuvent contenir :
- des unités hétérogènes,
- des valeurs manquantes,
- des hypothèses implicites propres aux sources originales.
